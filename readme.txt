Features Breakdown
For Users
Book Appointments

Add appointments by providing their name, email, phone, and selecting a service, date, and time slot.
Prevent duplicate appointments within a specified period (e.g., 2 days).
View Appointments

Users can view their booked appointments using their email or booking ID.
Cancel Appointments

Users can cancel their appointments before a specified deadline.
For Admin
View Appointments

Admins can view all appointments.
Filter appointments by:
User name
Email
Date
Update Appointment Status

Admins can confirm or cancel appointments.
Manage Services

Admins can add, update, or delete services offered by the engineering firm.
